# Blackjack Lite

A simplified command-line Blackjack game implemented in **C++**, with an
accompanying **Python** script for visualizing gameplay statistics.

This project features clean object-oriented design, file-based data
logging, and cross-language interaction between C++ and Python to
generate win/loss analytics.

------------------------------------------------------------------------

## Overview

Blackjack Lite simulates the core mechanics of Blackjack:

-   Standard 52-card deck (2--10, J/Q/K = 10, Ace = 11)
-   Player chooses to **Hit** or **Stand**
-   Dealer draws until reaching **17 or higher**
-   Winner determined by comparing totals without exceeding 21
-   Gameplay statistics are logged for later analysis

------------------------------------------------------------------------

##  Features

### **C++ Game Engine**

-   `Card` class (suit, rank, numeric value)
-   `Deck` class (vector of cards, shuffling, dealing)
-   `Game` class (game loop, win/loss logic, input handling)
-   Input validation for hit/stand decisions
-   File output:
    -   `game_log.txt` --- round-by-round summary\
    -   `win_stats.txt` --- cumulative win/loss tallies

### **Python Analytics**

-   Reads `win_stats.txt` generated by the C++ game
-   Produces bar charts (e.g., `results.png`)
-   Automatically runs after the game exits

### **Cross-Language Integration**

-   C++ writes statistics → Python reads them
-   Simple and extensible interface using plain text files

## Known Limitations

-   Ace always counts as 11\
-   Dealer hits until 17 only (no soft/hard logic)\
-   No betting or bankroll system\
-   Single-deck gameplay\
-   Text-only interface
